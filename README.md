# Hearts of Space Search API

FastAPI-based semantic search API for the
Hearts of Space archive project.

This service provides:

- semantic embedding search
- filtered metadata search
- “more like this” similarity search
- genre lookup
- program track retrieval

The API is used by the HOS web frontend, at https://hos-search.vercel.app.

---

## Architecture

```text
HOS API
    ↓
MySQL database
    ↓
program_embeddings table
    ↓
embeddings loaded into memory at startup
    ↓
semantic cosine similarity search
```

The service loads all program embeddings into memory
during startup for fast semantic search.

When new embeddings are generated by the
maintenance pipeline, the service must be restarted
to reload the cache.

## API Endpoints

### Genre List

```http
GET /genres
```

Returns all genres.

---

### Semantic Search

```http
POST /search
```

Request body:

```json
{
  "text": "dark ambient space music",
  "program_name": null,
  "genre": null,
  "program_content": null,
  "limit": 10
}
```

Supported filters:

- `text`
  Semantic embedding query.

- `program_name`
  SQL LIKE filter against program title.

- `genre`
  Exact genre match.

- `program_content`
  SQL LIKE filtering against:
  - descriptions
  - short descriptions
  - weather reports
  - track titles
  - artist names
  - album titles

- `limit`
  Maximum number of results.

---

### Similar Programs

```http
GET /similar/{program_number}
```

Returns embedding-similar programs.

Example:

```http
GET /similar/1421
```


## Notes

- The API currently loads all embeddings into memory.
  This is appropriate at current archive size.

- Cosine similarity is calculated in Python.

- Semantic search uses OpenAI embeddings rather than
  direct vector search inside MySQL.

- The service is intended for low-volume personal/public use,
  not large-scale production traffic.
